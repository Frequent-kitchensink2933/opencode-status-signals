## 📌 About the project

Opencode-status-signals helps you track your work sessions. It provides clear visual status updates for your coding tools. You see exactly what happens within your terminal and workspace. The tool maps themes to your current status. This keeps you informed about your workflow. You spend less time guessing and more time building.

## 🛠 Features

*   **Real-time tracking:** You receive live updates on your coding sessions. The status updates as you change tasks.
*   **Theme mapping:** You choose how your status signals appear. Match colors to your current mood or project type.
*   **Simple dashboard:** The interface displays information in one window. You read the status without extra menus.
*   **Low resource use:** The app runs in the background. It does not slow down your computer while you work.
*   **Clear indicators:** You see simple icons and color codes. These tell you if your session is active, idle, or syncing.

## ⚙️ Using the software

When the app opens, you see a small window. This window shows your current session status.

*   **Select a theme:** Click the "Settings" button. Choose "Appearance." Pick a theme from the drop-down list. The application applies the changes immediately.
*   **Customize signals:** Click on "Signals." You see a list of session types. Assign a color to each type.
*   **Check status:** The dashboard shows a status icon. A green icon means your session is active. A blue icon means the system waits for input. A red icon means the session is paused.

## 🆘 Troubleshooting common issues

Most users get the app running without trouble. If you face issues, try these steps:

**The app does not open.**
Restart your computer. Sometimes Windows needs a refresh to register new software. If this fails, delete the file and download it again from the link.

**The colors look wrong.**
Check your Windows "Apps & Features" settings. Ensure you have the latest version of the .NET runtime installed. Most Windows systems have this, but manual updates fix display errors.

**The app hides in the taskbar.**
Click the small arrow near your clock in the bottom right corner of your screen. You see the opencode-status-signals icon there. Right-click the icon to bring the window back to your desktop.

**Colors do not update.**
Click "Refresh" in the main menu. If the theme fails to change, restart the app. The config file might need a reset. Use the "Reset to Defaults" button in the Settings tab.

## 📝 Frequently asked questions

**Does this app capture my code?**
No. The app only tracks the status signals defined by your session parameters. It does not read your source code.

**Can I run multiple sessions?**
Yes. The app supports multiple views. Open the menu and click "Add Session" to create a new monitor window.

**Does this software cost money?**
No. This tool is free to download and use for your personal or professional projects.

**Can I minimize the app to the tray?**
Yes. Close the window, and the app continues to run in your system tray. Click the tray icon to reopen the main window.

**Where does the app save my settings?**
The installer keeps all files in the Program Files folder. Your personal preferences sit in your user profile folder under "AppData."
